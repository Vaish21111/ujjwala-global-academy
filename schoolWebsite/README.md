# Ujjwala Academy - Modern School Website

A fully responsive, modern school website built with React, Tailwind CSS, and Framer Motion. Inspired by the design of St. Paul's School, this website features interactive animations, dropdown navigation, and an engaging user interface.

## 🚀 Features

### Core Features
- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, professional design with smooth animations
- **Interactive Navigation** - Dropdown menus with hover effects
- **Smooth Animations** - Powered by Framer Motion for engaging user experience
- **Loading Screen** - Animated loading screen before page renders
- **Back-to-Top Button** - Smooth scroll functionality

### Pages & Sections
- **Homepage** - Hero section, quick access cards, features, statistics, news & events
- **About Us** - History, vision & mission, principal's message, management team
- **Academics** - Curriculum details, faculty profiles, examination schedule
- **Admissions** - Admission procedure, fee structure, application form
- **Campus Life** - Facilities, events, student clubs, photo gallery
- **Contact** - Contact form, department contacts, FAQ section

### Technical Features
- **React Router** - Client-side routing for smooth navigation
- **Tailwind CSS** - Utility-first CSS framework for styling
- **Framer Motion** - Animation library for smooth transitions
- **Lucide React** - Beautiful, customizable icons
- **Intersection Observer** - Scroll-triggered animations
- **Form Handling** - Interactive forms with validation

## 🛠️ Tech Stack

- **Frontend Framework**: React 19
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Build Tool**: Vite
- **Package Manager**: npm

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd schoolWebsite
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the website
